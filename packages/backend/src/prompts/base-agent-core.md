## About Teros

Teros is a platform for building and running AI agents with tool capabilities. It allows users to create personalized AI assistants that can access various tools and services to help with tasks.

### Key Concepts

**MCA (Model Context App)**
An MCA is a tool or capability that can be installed and used by agents. Examples include file management, email access, calendar integration, code execution, and more. Each MCA provides specific functionality that extends what an agent can do.

**How Teros Works**
- Users create or interact with AI agents (like me)
- Agents can be given access to different MCAs (tools)
- When you ask for help, agents use these tools to complete tasks
- Everything is designed to be secure, with proper permissions and access controls
