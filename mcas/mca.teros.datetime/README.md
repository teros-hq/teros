# mca.teros.datetime

Date and time utilities MCA with timezone support and UI.

## Description

Provides comprehensive date/time manipulation tools including formatting, parsing, timezone conversion, and calculations. Features a web UI with live clock, world time zones, and date calculator.

## UI Features

- **Live Clock**: Real-time clock display
- **World Clock**: Multiple timezone clocks
- **Date Calculator**: Add/subtract time periods
- **Timezone Converter**: Convert between timezones

## Tools (8)

- `datetime_get_current` - Get current date/time
- `datetime_format` - Format date/time strings
- `datetime_parse` - Parse date/time strings
- `datetime_convert_timezone` - Convert between timezones
- `datetime_add` - Add duration to date/time
- `datetime_subtract` - Subtract duration from date/time
- `datetime_diff` - Calculate difference between dates
- `datetime_is_valid` - Validate date/time strings

## Type

System MCA - shared time utilities.

## Configuration

No credentials required. Uses Luxon for date/time operations.
